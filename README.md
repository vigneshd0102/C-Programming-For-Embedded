# 🚀 C Programming for Embedded Systems

## 📌 Overview

This repository is a structured collection of C programs specifically designed for **Embedded Systems and Firmware Development**. It focuses on building strong fundamentals in low-level programming, which are essential for working with microcontrollers, memory, and hardware interfaces.

The repository is organized to cover everything from basic concepts to advanced embedded-oriented programming techniques.

---

## 🎯 Objectives

* Strengthen core C programming concepts
* Master **pointers and memory management**
* Understand **bit-level operations** for hardware control
* Build a foundation for **driver development and firmware design**
* Prepare for **Embedded Systems and Firmware Engineer roles**

---

## 🧠 Key Concepts Covered

* Variables, Data Types, and Control Structures
* Functions and Modular Programming
* Pointers and Pointer Arithmetic
* Arrays and Strings
* Structures and Unions
* Dynamic Memory Allocation
* Bit Manipulation (Set, Clear, Toggle, Masking)
* Register-Level Programming Concepts
* Memory Mapping Basics

## ⚙️ How to Run the Programs

1. Install a C compiler (e.g., GCC)
2. Clone the repository:

   ```
   git clone https://github.com/your-username/C-Programming-For-Embedded.git
   ```
3. Navigate to the file directory:

   ```
   cd Basics
   ```
4. Compile the program:

   ```
   gcc program_name.c -o output
   ```
5. Run the program:

   ```
   ./output
   ```
